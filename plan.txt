1. OVERVIEW
=======================================================================

A native Android app (Kotlin + Jetpack Compose) for controlling your-pax
cyberdeck via Bluetooth PAN. Inspired by Flipper Zero Android App's
modular architecture but with completely unique modern UI design.

The app communicates with your-pax via HTTP REST API over Bluetooth PAN
network (192.168.4.x). No cyberpunk theme - clean, modern, beautiful UI.

6. TECHNICAL ARCHITECTURE
=======================================================================

PATTERNS:
  - MVVM (Model-View-ViewModel)
  - Repository pattern (single source of truth)
  - StateFlow for reactive UI updates
  - Dependency Injection (Hilt / manual DI)

NETWORK LAYER:
  - Retrofit 2 + OkHttp
  - Base URL: http://192.168.4.1:8000/ (configurable)
  - Auth interceptor: adds Authorization + CSRF headers
  - Gzip interceptor for compressed responses
  - Timeout: 10s connect, 30s read

BLUETOOTH LAYER:
  - Android's BluetoothAdapter for discovery
  - BluetoothPan profile for PAN connection
  - BroadcastReceiver for BT state changes
  - NetworkUtils to detect IP assignment on pan interface

STATE MANAGEMENT:
  - ConnectionState: Disconnected | Scanning | Connecting | Connected | Failed
  - Each ViewModel manages its own UI state
  - Shared BluetoothConnectionState through Application class

ERROR HANDLING:
  - Network errors → show retry snackbar
  - API errors → parse error response, show toast
  - Connection lost → auto-reconnect / notify user
  - Offline mode → show cached data if available
